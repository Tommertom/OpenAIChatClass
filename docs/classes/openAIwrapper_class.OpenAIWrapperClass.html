runSpeechPrompt

  • runSpeechPrompt(input, model?, voice?, response_format?, speed?): Promise<OpenAIWrapperClass>

  • Runs a speech prompt using the OpenAI API.

    Parameters

    • input: string

      The input text for the speech prompt.

    • model: string & {} | "tts-1" | "tts-1-hd" = "tts-1"

      The model to use for generating the speech. Can be a string or one of the predefined values: "tts-1", "tts-1-hd".

    • voice: "echo" | "alloy" | "fable" | "onyx" | "nova" | "shimmer" = "alloy"

      The voice to use for generating the speech. Can be one of the predefined values: "alloy", "echo", "fable", "onyx", "nova", "shimmer".

    • response_format: "mp3" | "opus" | "aac" | "flac" = "mp3"

      (Optional) The format of the speech response. Can be one of the predefined values: "mp3", "opus", "aac", "flac".

    • Optional speed: number

      (Optional) The speed of the generated speech. A number between 0.1 and 3.0.

    Returns Promise<OpenAIWrapperClass>

    A Promise that resolves to the speech response.

    See

    • https://platform.openai.com/docs/guides/text-to-speech?lang=node
    • https://platform.openai.com/docs/api-reference/embeddings/create?lang=node.js
